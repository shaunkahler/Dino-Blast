# Dino Blast

https://github.com/shaunkahler/Dino-Blast/raw/master/wastedinmiami.mp4

A high-octane, neon-soaked side-scrolling shooter built with Rust and the [Macroquad](https://macroquad.rs/) game engine.

![Miami Vibes](https://img.shields.io/badge/Vibe-Miami%20Vice-pink)
![Rust](https://img.shields.io/badge/Made%20with-Rust-orange)

## Features

- **Retro Aesthetic**: Neon-drenched Miami sunset backdrop with parallax scrolling palms.
- **Iconic Enemies**: Battle against the legendary Chrome Dino and high-tech Gmail Helicopters.
- **Weaponry**: Fire rapid-shot projectiles and rare "Gmail" power-shots.
- **Dynamic Gameplay**: Physics-based movement, jumping, and mouse-aiming combat.
- **Pixel Perfect**: Scalable virtual resolution (1280x720) with a CRT-style nearest-neighbor filter.

## Controls

- **Move**: `A` / `D` or `Left` / `Right` arrows.
- **Jump**: `W`, `Space`, or `Up` arrow.
- **Aim**: Use the `Mouse` to aim your weapon.
- **Shoot**: `Left Mouse Button`, `J`, or `K`.
- **Restart**: `R` or `Space` (when Game Over).

## Getting Started

### Prerequisites

You'll need the Rust toolchain installed. If you don't have it, get it at [rustup.rs](https://rustup.rs/).

### Running the Game

```bash
cargo run
```
